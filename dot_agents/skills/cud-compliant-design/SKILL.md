## 2. 画面用の推奨配色例
- 画面用の比較的見分けやすい組み合わせは以下の通りです。
  - 特に、「赤・緑・青・空色」の4色セットは有用です。
  - というのも、3次元の量をプロットする場合、多くの場合はX軸・Y軸・Z軸に赤・緑・青を割り当て、空色を補助的に使うことで、視認性が高まるためです。
  - たとえば、X, Y, Zに加えてそのノルムを空色で表現する、などの用途が考えられます。

### アクセントカラーのみを使った4色の場合
- 赤, 黄色, 緑, 空色
- 赤, 黄色, 青, 空色
- 赤, 緑, 青, 空色
- オレンジ, 黄色, 紫, 空色
- オレンジ, 緑, 紫, 空色
- 黄色, 紫, 空色, ピンク

## 3. 線種・マーカーの使い分け（折れ線グラフ・時系列プロット）

色だけに頼らず、**線種（linestyle）やマーカー形状も系列ごとに変える**こと。これにより、モノクロ印刷時や色覚特性に関わらず系列を区別できる。

### 線種の割り当て順序

後にプロットされる系列ほど**疎な線種**を使う。線が重なった際、疎な線の隙間から密な線が見えるため視認性が向上する。

| 優先順 | 線種 | matplotlib `linestyle` | 説明 |
| :--- | :--- | :--- | :--- |
| 1（最初） | 実線 | `"-"` | 最も密。下に隠れても疎な線の隙間から見える |
| 2 | 破線 | `"--"` | 中程度の疎さ |
| 3 | 点線 | `":"` | 最も疎。最前面でも下の線が見える |
| 4 | 一点鎖線 | `"-."` | 4系列以上の場合に使用 |

### マーカーの活用

散布図やデータ点数が少ない場合は、マーカー形状も系列ごとに変える（`"o"`, `"s"`, `"^"`, `"D"` など）。

### 線幅

デフォルトよりも1.5倍程度太めにプロットすると視認性が向上する（例: matplotlib デフォルト 1.5 → 2.0〜2.5 程度）。

### 例: 3次元量（X, Y, Z）のプロット

```python
STYLES = [
    {"color": "#FF4B00", "linestyle": "-",  "linewidth": 2.0, "label": "x"},  # 赤・実線
    {"color": "#03AF7A", "linestyle": "--", "linewidth": 2.0, "label": "y"},  # 緑・破線
    {"color": "#005AFF", "linestyle": ":",  "linewidth": 2.0, "label": "z"},  # 青・点線
]
for i, s in enumerate(STYLES):
    ax.plot(time, data[:, i], **s)
```
